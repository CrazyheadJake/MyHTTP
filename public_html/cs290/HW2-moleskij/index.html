Web Dev Concepts

Definition of Terms:

Website - A website is an html file that is displayed on your web browser.

Web Application - A web application is a program that is designed to run on a website.

Web Browser - A web browser is a program that can render html files for you as well as send and receive HTTP requests.

Web Server - A web server is a computer that hosts a website (or multiple websites) on it.

HTTP Request - An HTTP request is a network packet that is sent to a web server in order to get a response of some sort.

HTTP Response - An HTTP response is a network packet that is sent to the client, usually in response to an HTTP request.

Frontend Design:

Frontend design is about making the most appropriate experience for the users who use your product. This includes visual elements as well as the interactive experience with your website. Things like color scheme, font, photos, and navigation systems all affect the user experience.

Effective
How effective is the site at meeting the users requirements and will they end up with the correct results.
Efficient
How quickly and easily can the user accomplish what they need to? Do the results appear quickly?
Easy to Navigate
Will the user understand how to reach their goal, even if they've never been to the site before? Will they remember how to do it next time?
Error-free
Will it be available and accessible to all without any problems? Will the user run into issues when accomplishing their goals?
Enjoyable
Will the website fit the unique needs of the user. Will the user go back to the website and use it again? Are they satisfied with the previous four E's?

Page layout tags break up the flow of content. They separate different sections of content from another, often when the two sections are for two different purposes. Some examples of these tags are header, nav, main, section, article, and footer. These help define where items are displayed on the page.

Tag Description
p The p tag is used to denote a paragraph of text. It always starts on a new line and leaves a newline below it.
a The a tag is used to denote a link that can link to another page on the website or another website entirely.
ul The ul tag is used to create a bullet point list of items.
ol The ol tag is used to create a numbered list of items.
h1 The h1 tag is used to create header text which is larger than other text and bolded.
nav The nav tag is a page layout tag. It is used to denote a group of links that will take you to other pages or other places on the same page.
